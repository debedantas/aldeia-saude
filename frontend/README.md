# React Frontend - Aldeia Saúde

## Visão Geral

Frontend desenvolvido em React com TypeScript, Vite e Tailwind CSS para o dashboard de gestão de saúde.

## Credenciais de Demo

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`
